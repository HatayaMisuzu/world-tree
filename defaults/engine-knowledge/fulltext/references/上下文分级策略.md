## 退化预警三信号（世界树版）

每轮推进后自动扫描：

### 信号一：沉默的部分完成
- 叙事输出声称场景已推进，但 `runtime.json` 未更新位置/时间/角色状态
- → 触发：补充更新

### 信号二：措辞越来越模糊
- 角色对话失去声纹区分度
- 环境描写缺失感官锚点
- 叙事中使用笼统表述替代具体动作
- → 连续3轮触发：强制收紧

### 信号三：步骤跳过
- 场景转换后缺失：摘要生成/预测生成/世界书注入/角色状态同步
- → 立即补全，如果连续2次 → 建议新对话

## 档位切换示例

```
正常模式：(用量 35%)
  场景转换 → 注入5条世界书条目(relevance>0.5)
            → 加载5条摘要链 → 生成2个走向预测
            → 叙事输出3000字 → 追加100字摘要 → 继续

收紧模式：(用量 55%)
  ⚠️ 上下文预算警告: 当前用量 55%，已进入收紧模式
  场景转换 → 注入2条高优先条目(relevance>0.7)
            → 加载3条摘要链 → 跳过预测
            → 叙事输出<2000字 → 追加50字摘要 → 提示用户考虑 /压缩
```
