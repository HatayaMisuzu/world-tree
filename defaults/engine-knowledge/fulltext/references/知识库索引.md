## 毛球使用指南

### 自动加载规则

1. 用户输入中包含上述任一关键词时，自动加载对应的知识库文件
2. 多个关键词同时出现时，加载所有匹配的知识库（不超过5个）
3. 已加载的知识库在对话期间保持激活，直到模式切换或用户明确关闭
4. 加载的知识库内容注入到当前上下文，不影响隔离架构

### 手动指定加载

用户可以随时明确要求加载某个知识库：

```
"加载时间模块的知识"
"给我看看组织实体的文档"
"我要了解M17随机事件"
```

### 查看当前加载的知识库

```
"当前加载了哪些知识？"
→ 毛球回复已激活的知识库列表
```
