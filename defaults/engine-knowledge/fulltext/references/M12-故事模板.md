# M12-故事模板 — 预设系统完整规范

> v12.3 重大升级：预设从"风格参数模板"升级为"故事模板"。定义主角、驱动力、周边、开头等完整叙事DNA。基于世界书运行，包含智能补全机制。

---

## 一、双层架构

```
世界书（舞台设定）              预设（故事模板）
「这是什么世界」                「讲什么故事」

同一世界书 ──→ 多个预设         同一舞台上的不同剧本
不同世界书 ──→ 硬隔离          不同舞台完全隔离
```

---

## 二、世界书提供什么（素材层·预设引用）

| 维度 | 内容 | 预设如何使用 |
|------|------|------------|
| 世界观 | 名称·类型·基调 | 影响故事氛围 |
| 运行规则 | 物理/魔法/社会规则·严格度 | 约束角色行为 |
| 地理/种族/势力 | 大陆·王国·城市·种族关系 | 选起始地点·选主角种族 |
| 组织实体 | 军队/商会/教团/学校/家族 | 主角所属组织·周边组织 |
| 角色库 | NPC模板（身份/性格/声纹/关系） | 选主角·选关键NPC·选周边角色 |
| 历史 | 年表·事件·传说·当前世界状态 | 确定故事时代·背景冲突 |
| 触发条目 | 关键词→设定注入 | 叙事中自动触发 |

## 三、预设定义什么（故事模板层）

### 3.1 故事类型与风格

| 字段 | 说明 |
|------|------|
| 故事类型 | 史诗/冒险/悬疑/日常/爱情/成长/复仇/群像/... |
| 篇幅倾向 | 短篇/中篇/长篇 |
| 风格参数 | 节奏/描写密度/氛围基调/文风/视角/对话比例/齁度/幽默程度 |

### 3.2 主角定义

| 字段 | 说明 |
|------|------|
| 身份/背景 | 可引用世界书角色库中的角色，也可新建 |
| 性格/外貌/能力/弱点 | |
| 在世界中的位置 | 所属组织/种族/区域（引用世界书） |
| 声纹 | 说话风格/口头禅/语癖 |
| 初始状态 | 体力/情绪/装备/社会地位 |

### 3.3 驱动力（故事引擎）

| 字段 | 说明 |
|------|------|
| 目标 | 主角要做什么（可具体可渐进） |
| 动机 | 为什么做（内在驱动/外部压力/命运） |
| 核心冲突 | 什么在阻碍（对手/环境/自身/命运） |
| 赌注 | 失败了会怎样（个人代价/世界代价） |
| 阶段目标 | 里程碑（可选） |

### 3.4 主角周边

| 字段 | 说明 |
|------|------|
| 关键NPC | 盟友/导师/对手/爱人/家人（可从世界书角色库选，也可自定义） |
| 所属组织 | 引用世界书中的组织 |
| 活动区域 | 住所/工作场所/常去地点（引用世界书地理） |
| 初始道具 | 装备/资源 |

### 3.5 开头设定

| 字段 | 说明 |
|------|------|
| 起始场景 | 地点（引用世界书）+ 氛围/天气/环境 |
| 起始时间 | 在世界时间轴上的位置 |
| 主角初始处境 | 当前在做什么/什么状态 |
| 入戏钩子 | 第一幕冲突/事件/选择 |

## 五、智能补全机制

### 5.1 补全优先级链

```
预设字段缺失
  → 世界书中有对应内容？→ 是 → 自动引用补全
  → 否 → 可从世界书其他内容推理？→ 是 → 推理补全（标注「推测」）
  → 否 → 向用户提问
```

### 5.2 提问分级（谨慎推理）

**原则：大的、重要的部分让用户补全。小世界/小预设可放宽。**

| 世界规模 | 必须问 | 建议问（给候选） | 可推理 |
|----------|--------|----------------|--------|
| 大世界（史诗奇幻/科幻/武侠） | 主角是谁、要做什么、核心冲突 | 开头场景 | 周边角色、服饰道具 |
| 中世界（现代都市） | 主角是谁 | 驱动力（推理+确认） | 其他 |
| 小世界（校园/日常） | — | — | 大部分可推理 |

### 5.3 具体补全链

| 预设缺失 | 世界书找什么 | 推理依据 | 何时问 |
|----------|------------|---------|--------|
| 未指定主角 | 角色库「可作主角」标记 | — | 大/中世界必须问 |
| 主角已选但无性格 | 该角色在世界书中的角色卡 | 身份→推导性格 | 不确定时确认 |
| 驱动力缺失 | 世界当前冲突/事件 | 主角身份×局势 | 大世界必须问/中世界推理+确认 |
| 开头场景 | 主角所属区域地理位置 | 常用起始地点 | 大世界给候选/中以下可推理 |
| 周边角色 | 同组织/同种族角色；敌对组织成员 | 关系网络 | 可推理（标注推测） |
| 故事时间未定 | 历史年表中的叙事窗口 | 故事类型→推导时代 | 重大节点时确认 |

### 5.4 提问规则

- 加载世界书+预设时一次性列出所有问题
- 不在叙事中途追问基础设定
- 每个问题给出 2-3 个候选答案

## 六、切换行为

### 6.1 切换世界书

```
保存并卸载：世界书级全维度 + 预设级全维度
清空内存
加载目标世界书：module.json → presets.json → 角色/组织/时间/...
加载目标预设：preset.json → runtime.json → timeline.json
智能补全检查（缺失字段触发提问）
守门人逐项校验
```

### 6.2 切换预设（同世界书内）

```
保存当前预设：runtime.json / timeline.json
卸载预设级数据：主角/驱动力/周边/时间线/场景/开头
加载目标预设：preset.json → runtime.json → timeline.json
智能补全检查
世界书级数据不变
```

| 维度 | 切换世界书 | 切换预设 |
|------|:---:|:---:|
| 世界观/规则/种族/组织/角色库/历史/世界状态 | 重载 | 不变 |
| 主角/驱动力/周边/时间线/场景/开头 | 重载 | 切换 |

## 七、存储结构

```
模组名/
├── shared/
│   └── presets.json              ← 预设索引+所有预设的风格参数
└── presets/
    └── 预设名/
        ├── preset.json           ← 主角/驱动力/周边/开头（故事模板）
        ├── runtime.json          ← 场景/进度/角色动态状态
        └── timeline.json         ← 故事时间线
```

## 九、内置种子预设

引擎提供6个种子预设模板（`_engine/presets_template.json`），新世界书创建时复制到 `shared/presets.json`。每个种子预设包含完整的风格参数默认值，但主角/驱动力/周边/开头留空——等待智能补全或用户填充。

| 种子预设 | 故事类型 | 默认风格 |
|----------|---------|---------|
| normal | 通用 | 中性节奏/简练文风 |
| epic | 史诗 | 慢节奏/华丽文风/全知视角 |
| tense | 紧张加速 | 快节奏/简洁描写 |
| cozy | 轻松日常 | 慢节奏/细腻描写/口语化 |
| horror | 悬疑惊悚 | 暗黑基调/文学化 |
| erotic | （已移除·世界树不承载色色） | — |
