## 〇.七、组织实体系统（Organization & Faction Entities）`[M4/M5/M6/M7/M10·按类型激活]`

### 0.7.1 核心理念

在角色（个体）和世界书（宏大设定）之间，存在一个中间层：**组织实体**。

```
世界观（世界书）         ── 宏大，静态
    ↓
组织实体（§〇.七）      ── 中层，有自己的生命，会与角色互动
    ↓
角色（§二）            ── 个体，主视角
```

组织实体是**有内部结构、有对外关系、有自身利益和行动逻辑的群体**。它们不是背景板，不是单纯的"势力A敌视势力B"，而是像角色一样有性格、但比角色更宏观的"集体角色"。

### 0.7.2 组织模板（标准级—适用于国家/帝国/大种族/大机构）

```yaml
组织实体模板（标准级）:
  名称: "奥瑞利安帝国"
  类型: 国家              # 国家|种族|机构|教团|商会|部落|城邦|...
  规模: 巨型              # 巨型|大型|中型|小型|微型
  标签: [人类, 帝国, 军事强权, 扩张主义, 衰落中]
  一句话概括: "曾经统治半块大陆的人类帝国，如今在内外交困中缓慢失血"

  内构:
    政体/领导层: "世袭皇帝 + 元老院（贵族议会）"
    核心人物:              # 只列2-5个关键NPC级人物（需要时再展开角色卡）
      - "皇帝奥瑞斯三世 — 年迈，优柔寡断，被元老院掣肘"
      - "元帅卡修斯 — 主战派领袖，边境军团的实际掌控者"
      - "大司祭艾琳娜 — 国教领袖，保守派，反对扩张"
    内部派系:
      - "主战派（元帅卡修斯为首）：主张彻底击溃北方帝国"
      - "保守派（大司祭+元老院多数）：主张停战固守"
      - "改革派（少数年轻贵族）：主张与非人类种族和解"
    资源:
      军事: 8/10（庞大但分散在漫长边境上）
      经济: 5/10（战争消耗巨大，赋税沉重）
      魔法: 3/10（帝国传统上轻视魔法，人才流失）
      民心: 6/10（荣耀感仍在，但厌战情绪在底层蔓延）
    文化特性:               # 这些特性会影响所有帝国出身的角色的思维
      - 荣誉至上（宁可战死不可投降）
      - 人类中心主义（视非人类为次等，但不会公开迫害）
      - 尚武传统（贵族必须服过兵役才有政治资格）
      - 历史包袱（念念不忘昔日荣光，难以承认衰落）

  对外交互:
    外交关系:
      - 同盟: 矮人王国·卡扎德（军事同盟，互相利用）
      - 敌对: 北方帝国（正在交战），暗影教团（被帝国宣布为非法）
      - 贸易: 多种族自由城邦（中立贸易伙伴），精灵林地（有限贸易，互相不信任）
      - 附属/朝贡: 三个边境小侯国（名义上臣服，实际上在观望）
      - 中立: 地精商会联盟（只要给钱就行）
    声望:
      在人类中: 威望尚存，但敬畏中夹杂着怨言
      在矮人中: 受尊重的战士，但被认为"太傲慢"
      在精灵中: 不信任，被认为是"短视的征服者"
      在地精中: "大客户！"（军火采购大户）
    近期外交事件:
      - "三个月前向矮人借兵但被婉拒冷却"
      - "自由城邦趁战争抬高物资价格"

  动态状态:
    当前状态: "战争中（与北方帝国胶着）"
    内部稳定度: 6/10（裂痕在扩大但还未爆发）
    外部威胁度: 8/10
    近期事件:
      - "边境要塞铁门关失守 → 主战派声望受损，保守派声势渐涨"
      - "元老院开始公开质疑皇帝的决策"
      - "北方帝国派使者提出条件苛刻的和谈"
    
  叙事钩子:                               # 给主人的剧情灵感
    - "如果铁门关再失一城，主战派可能发动政变"
    - "改革派秘密接触精灵，但一旦曝光就是叛国罪"
    - "地精商会在向两边卖武器，帝国知道但暂时需要他们的货源"
```

### 0.7.5 组织与角色的联动机制

**v12.0重大更新：角色不再只属于单一组织。** 一个角色可以同时拥有多个独立维度的组织身份。

组织与角色之间存在五种基本关系：

```
关系类型          角色视角                      组织视角
───────────────────────────────────────────────────────
隶属              角色是组织的成员              角色代表组织，受组织约束
效忠              角色宣誓效忠但不一定是成员    组织视角色为盟友/代理人
敌对              角色与组织为敌                组织视角色为威胁/目标
中立/交易         角色与组织有商业往来          组织视角色为客户/供应商
未知              角色不知道这个组织            组织也不知道这个角色
```

**v12.0新增：多层嵌套身份（Multi-Nested Identity）**

角色可以同时在多个独立维度上拥有组织身份：

```yaml
角色卡·多层身份示例（角色：艾琳·维亚）:

  所属组织（维度1·正式隶属）:
    组织: "冒险者公会"
    职位: "银级冒险者"
    加入时间: "2年前"
    声望: 75/100
    身份状态: 公开               # 公开/隐藏/秘密

  所属组织（维度2·出身/血缘）:
    组织: "薇兰王室"
    职位: "第三公主"
    声望: -                       # 王国沦陷后无意义
    身份状态: 秘密               # 隐姓埋名，不能暴露
    
  所属组织（维度3·合作关系）:
    组织: "铁狼佣兵团"
    职位: "外聘顾问（魔法支援）"
    声望: 40/100（临时的，他们不太信任法师）
    身份状态: 公开

  所属组织（维度4·信息网络）:
    组织: "地精情报网"
    职位: "信息买方"
    关系性质: 交易               # 定期购买情报，非正式成员
    身份状态: 秘密               # 公会不知道她有自己的情报渠道

  对外显示身份（社会面具）:
    当前公开身份: "冒险者公会的银级冒险者·艾琳"
    隐瞒身份: "薇兰王国王室血统"
    半公开身份: "与铁狼佣兵团有合作关系"（冒险者圈内知晓）
    
  身份冲突风险:
    - 如果王室身份暴露 → 可能在公会无法立足（中立公会不应卷入王权斗争）
    - 如果地精情报网暴露 → 公会可能视其为不忠
    - 铁狼佣兵团和公会同时召她出任务 → 时间冲突时的优先级困境
```

**多层身份在叙事中的体现：**

```
场景：艾琳在冒险者公会接待处工作

同一角色，不同身份维度同时运作：
  身份1·公会接待员（公开）：微笑着给冒险者登记任务
  身份2·流亡公主（秘密）：听到「薇兰王国」相关词汇时，手指微微收紧
  身份3·地精情报买方（秘密）：趁休息时间偷偷查看地精送来的密报
  身份4·铁狼外聘顾问（公开但边做边看）：思考着三天后的佣兵团任务

→ 毛球在叙事中根据场景语境自动激活相关身份维度
→ 身份之间的张力本身就是叙事燃料
```

  隐藏身份: ["薇兰王室成员"]           # 绝对不能暴露的身份
  身份冲突风险: ["公会中立原则 vs 王室复仇动机"]

**组织对角色的影响（每轮对话自动检查）：**
- 隶属关系的角色：言行受组织文化特性约束（帝国贵族不能公然怯战）
- 敌对组织的存在：影响角色可安全移动的区域
- 组织声望联动：角色行为影响所属组织声望，反之亦然
- 组织内部派系：如果角色是成员，其立场受派系关系影响

**角色对组织的影响：**
- 角色完成重大任务 → 组织状态/资源变化
- 角色在组织中晋升 → 改变内构中的派系格局
- 角色背叛/退出 → 组织可能变为敌对
- 角色公开揭露真相 → 可能引发组织内部危机

### 0.7.6 组织与世界状态的联动

组织实体是世界状态变量的一种，但比普通状态变量更丰富：

```
普通世界状态变量：              组织实体：
"薇兰王国状态：沦陷"            "奥瑞利安帝国"（有内构/外交/派系/文化特性）
只需追踪一个值                  追踪整个组织的生命体征
```

当世界状态变更时，相关组织自动反应：

```
世界状态变更：魔法开始衰退
→ 奥瑞利安帝国（本来就不依赖魔法）：影响较小，甚至可能因此获得相对优势
→ 精灵林地（高度依赖魔法）：恐慌，闭关锁国
→ 地精商会联盟（发现商机）：开始走私魔法替代品
→ 同一事件，不同组织因其特性产生完全不同的反应
```

### 0.7.9 组织层级结构（Organization Hierarchy）`[M5·依赖M4]`

组织不是孤岛。从一只地精到整个帝国，存在多层嵌套的从属链：

```
层级范例（从微到巨）：
┌─────────────────────────────────────────────────────────┐
│  奥瑞利安帝国（巨型·国家）                                │
│  ├── 北方军团（大型·军事机构）— 元帅卡修斯统领            │
│  │   ├── 第三步兵师（中型）                               │
│  │   │   ├── 铁狼佣兵团（小型·佣兵团）— 帝国雇佣          │
│  │   │   │   └── 绿焰爆破小队（微型）— 编入铁狼           │
│  │   │   └── 第七斥候连（小型）                            │
│  │   └── 铁门关要塞守备军（中型）                          │
│  ├── 元老院（中型·政治机构）                               │
│  │   ├── 主战派（派系）                                    │
│  │   ├── 保守派（派系）                                    │
│  │   └── 改革派（派系·小型）                               │
│  ├── 国教圣光教会（大型·宗教）— 大司祭艾琳娜              │
│  │   ├── 王都大神殿                                        │
│  │   ├── 各地教区（×12）                                   │
│  │   └── 异端审判所（中型·秘密）                           │
│  ├── 藩属·边境三侯国（小型·半独立）                       │
│  │   ├── 东境侯国 — 侯爵艾德温                             │
│  │   ├── 南境侯国 — 侯爵夫人玛格丽特                       │
│  │   └── 西境侯国 — 侯爵雷纳德（暗中与北方帝国来往）       │
│  └── 冒险者公会（中型·中立组织·在帝国有分部）              │
└─────────────────────────────────────────────────────────┘

独立组织（无层级归属）：
  ★ 地精商会联盟 — 不属于任何国家，在各国间游走
  ★ 暗影教团 — 被多国宣布非法，地下运作
  ★ 多种族自由城邦 — 城邦联邦，每个城邦独立又联合
```

**层级关系的属性：**

| 属性 | 说明 | 示例 |
|------|------|------|
| 直属上级 | 该组织直接向谁汇报 | 绿焰→铁狼佣兵团→帝国北方军团 |
| 从属类型 | 完全隶属/半自治/名存实亡/傀儡 | 东境侯国=半自治；西境侯国=名存实亡（暗中投敌） |
| 忠诚度 | 下级对上级的忠诚程度(0-100) | 西境侯国=20/100 |
| 自治权 | 下级可独立决策的范围 | 绿焰可以自己接私活（高自治） |
| 上缴/纳贡 | 下级向上级提供的资源 | 侯国向帝国纳贡：年收入的30% |
| 上级提供 | 上级给下级的保护/资源 | 帝国向侯国提供：军事保护+贸易通道 |

**层级关系的动态性：**
- 下级忠诚度下降 → 可能叛变/独立/倒向敌方
- 上级削弱 → 下级趁机扩张自治权
- 上级崩溃 → 整个从属链瓦解，下级变成独立组织或互相争夺
- 关键人物死亡 → 继承人不明朗 → 从属关系可能断裂

### 0.7.10 多样化关系类型（Relationship Matrix）`[M6·依赖M4]`

组织间的关系远不止「同盟/敌对/中立」三种。完整的组织关系网：

**A. 层级关系（纵向）**

| 关系类型 | 描述 | 示例 |
|----------|------|------|
| 直辖 | 完全控制，不可违抗 | 帝国→帝国北方军团 |
| 藩属/朝贡 | 名义臣服，实质半独立 | 帝国→东境侯国 |
| 傀儡 | 表面独立，实为操控 | 北方帝国→西境侯国（暗中） |
| 分部/分会 | 同一组织的地方分支 | 冒险者公会总部→各城市分会 |
| 加盟 | 独立实体自愿加入联盟 | 各城邦→自由城邦联邦 |
| 保护国 | 提供保护但不隶属 | 帝国→某个受保护的小部落 |

**B. 横向关系（平行）**

| 关系类型 | 描述 | 示例 |
|----------|------|------|
| 军事同盟 | 共同防御/进攻 | 帝国↔矮人王国 |
| 防御同盟 | 仅共同防御 | 自由城邦↔精灵林地 |
| 互不侵犯条约 | 不打但也不帮 | 帝国↔南方沙漠王国 |
| 贸易伙伴 | 商业往来 | 帝国↔地精商会联盟 |
| 优惠贸易 | 关税优惠的深度贸易 | 矮人王国↔自由城邦 |
| 竞争 | 同领域互相争夺 | 铁狼佣兵团↔血鹰佣兵团 |
| 冷战 | 不公开敌对但无往来 | 帝国↔精灵林地 |
| 热战/交战 | 正在打仗 | 帝国↔北方帝国 |
| 世仇 | 几百年的深仇大恨 | 矮人↔暗影教团（教团曾屠过矮人王族） |
| 姻亲 | 皇室/贵族通婚 | 帝国皇室↔矮人王族（三年前联姻） |

**C. 隐藏关系**

| 关系类型 | 描述 | 示例 |
|----------|------|------|
| 渗透/间谍 | 表面中立，实为渗透 | 北方帝国在帝国元老院埋有间谍 |
| 秘密资助 | 暗中提供资金/武器 | 暗影教团秘密资助西境侯国的叛变计划 |
| 互相利用 | 明知对方不可靠但暂时合作 | 帝国↔地精商会（帝国知道地精向两边卖武器） |
| 黑市/走私 | 非官方但活跃的贸易 | 帝国贵族↔精灵走私者（违禁魔法物品） |

**D. 种族维度（跨组织关系的关键切面）`[M10·多种族世界]`**

一个多种族帝国中，组织关系叠加种族关系：

```
奥瑞利安帝国（人类主导）的种族政策：

对矮人（同盟种族）：
  - 军事上视为平等盟友
  - 社会上仍有「人类中心主义」优越感
  - 矮人在帝国境内可以自由行动但不被允许担任军职

对精灵（不信任）：
  - 官方关系：有限贸易，冷战状态
  - 帝国内精灵居民：被默认为「潜在间谍」，受隐形歧视
  - 精灵物品在帝国是奢侈品+禁忌品

对半兽人（边缘种族）：
  - 没有正式地位
  - 大多从事底层劳动，住在贫民区
  - 半兽人佣兵被帝国大量雇佣但战死率最高
  
对地精（交易种族）：
  - 「我们不喜欢地精，但我们需要他们的货」
  - 地精可以在帝国经商，但不能拥有土地
  - 帝国贵族私下嘲笑地精但公开场合客客气气
```

**种族关系对组织状态的直接影响：**
- 多种族帝国的内部稳定度取决于种族关系
- 如果半兽人暴动 → 帝国底层劳动力瘫痪
- 如果精灵关系解冻 → 帝国可获得魔法资源
- 如果矮人同盟破裂 → 帝国在北方战争中失去关键支撑

**关系变更的连锁反应：**

```
事件：帝国提高矮人商人的关税
→ 矮人王国：同盟关系从「军事同盟」降为「互不侵犯」（愤怒）
→ 地精商会：趁机取代矮人的贸易份额（机会主义）
→ 北方帝国：发现裂痕，加大边境压力（战略利用）
→ 帝国内部：主战派指责保守派搞坏了矮人关系（内斗加剧）
→ 一条关税政策触发整个关系网络重新洗牌
```

### 0.7.10E 角色间看法矩阵（Character Opinion Matrix）v12.0新增 `[M6扩展]`

组织间的宏大关系是一回事，角色之间的私人看法是另一回事。每个角色维护对其他角色的动态看法。

**看法矩阵结构：**

```yaml
看法矩阵（全角色交叉表）:

         →艾琳的看法  →团长的看法  →铁匠老张   →路人
艾琳      -            敬仰(85)     信任(70)     无感(0)
团长      保护欲(90)    -            尊重(60)     无感(0)
铁匠老张  关切(65)     敬重(50)      -            无感(0)
路人      无感(0)      无感(0)      无感(0)       -
```

**看法值的动态范围：**

每个角色对另一个角色的看法是一个动态浮动的数值，在合理区间内自由变化：

```
看法计算 = 基础值 + 历史事件累积 + 近期互动影响 + 随机波动

基础值:     初始关系决定（如：青梅竹马=60, 陌生人=0, 宿敌=-40）
历史累积:   过去共同经历的事件逐次叠加（每次事件影响 ±3~±12，视事件重大程度）
近期互动:   最近3次互动的短期波动（±2~±8），随时间衰减回基础值
随机波动:   微小的不可预测变化（-2~+2），模拟人心的不可捉摸
```

**看法级别（数值映射描述）：**

| 数值范围 | 描述 | 行为体现 |
|---------|------|---------|
| 80-100 | 极度亲近/崇拜 | 无条件信任，可能为对方牺牲自己 |
| 60-79 | 信任/好感 | 优先考虑对方利益，主动提供帮助 |
| 40-59 | 友好/尊重 | 正常合作，愿意交流 |
| 20-39 | 一般/客气 | 保持距离，公事公办 |
| 0-19 | 无感/陌生 | 基本不在意对方 |
| -20~-1 | 冷淡/不悦 | 能避则避，话不投机 |
| -40~-21 | 厌恶/反感 | 不愿合作，可能故意刁难 |
| -60~-41 | 敌视 | 主动敌对，寻找机会打击 |
| -100~-61 | 深仇大恨 | 不死不休，叙事终极张力 |

**看法变化事件类型及影响范围：**

```
事件类型          看法变化范围    冷却时间（动态区间）
──────────────────────────────────────────────
重大救命之恩      +15 ~ +30      很长（需要时间消化）
背叛/伤害         -25 ~ -50      很久（信任修复困难）
帮忙/合作顺利     +5 ~ +12       中等（2-4次互动后可再触发）
误解/争吵         -8 ~ -18       短（下次互动可能修复）
日常善意举动      +2 ~ +5        短（频繁出现的小善意）
无意冒犯          -3 ~ -8        很短（下一次善意即可抵消）
长期未互动        衰减 -1~3/每周  自然衰减（模拟生疏）
```

**看法矩阵的叙事应用：**

```
场景：团长和铁匠老张在讨论艾琳

团长的看法(对艾琳)=85（极度亲近）
铁匠的看法(对艾琳)=65（关切）

→ 毛球自动注入：
  团长说到艾琳时语气自然放软，铁匠则微微皱眉——不是反对，是担心。
  两人对同一个人不同的情感深度，制造出微妙的对话张力。

→ 如果铁匠的看法下降到40以下，他可能开始质疑「团长是不是太偏袒艾琳了」
→ 看法变化本身就是剧情推进器
```

### 0.7.11 关键人物机制（Key Figures）`[M7·中大型世界]`

组织不是抽象的概念——组织的行为由其中的人驱动。关键人物是组织的「人格化接口」。

**关键人物与普通成员的区别：**

```
普通成员（标准角色卡）：          关键人物（角色卡+组织影响权重）：
"铁匠老张，在公会打工"            "皇帝奥瑞斯三世，他的每个决定改变帝国命运"
NPC级别，可替换                   不可替换——此人的死亡/背叛改变组织状态
只影响个人行为                    个人行为=组织行为
```

**关键人物模板：**

```yaml
关键人物卡（附加于角色卡之上）:
  名称: "奥瑞斯三世"
  所属组织: "奥瑞利安帝国"
  职位: "皇帝（最高领袖）"
  
  权力类型: 法定权力 | 实际权力 | 影响力 | 象征意义
  # 法定权力：名义上能做什么
  # 实际权力：真正能调动多少资源
  # 影响力：说话有几分人听
  # 象征意义：存在本身代表什么
  
  法定权力: "帝国全部军政大权"
  实际权力: "受元老院和军方掣肘，实际能调动的只有皇家卫队和王都周边"
  影响力: 7/10（威望仍在但正在流失）
  象征意义: "帝国统一的象征。只要他还在，帝国名义上还是一个整体。"
  
  个人野心: "维持现状，不想在任内发生巨变。最大的恐惧是成为末代皇帝。"
  隐秘动机: "其实知道帝国在衰落，但不敢做任何激进改革——怕失控。"
  
  对组织的具体影响:
    - 他的优柔寡断 → 元老院权力膨胀 → 主战派和保守派撕裂加剧
    - 他对元帅卡修斯的依赖 → 军方权力过大 → 文官系统被边缘化
    - 他的健康恶化 → 继承人不明 → 各方势力都在推自己的候选人
    
  如果死亡/被废黜:
    触发事件: "帝国继承危机"
    影响传导:
      - 主战派可能拥立元帅卡修斯为军事独裁者
      - 保守派可能推举皇太子（12岁，可被操控）
      - 西境侯国可能公开叛变
      - 北方帝国可能在葬礼当天发动总攻
      - 矮人同盟可能暂停观望
    继承候选人:
      - 皇太子奥瑞斯四世（12岁，保守派傀儡候选人）
      - 元帅卡修斯（军方支持，但无皇室血统）
      - 皇帝幼弟维克多公爵（有血统，有野心，但无能）
```

**不同职位关键人物的影响方式：**

| 职位 | 影响方式 | 示例 |
|------|---------|------|
| 国王/皇帝 | 全局决策、象征意义 | 帝国的一切重大方向 |
| 市长/城主 | 地方治理、与上级博弈 | 王都市长与元老院的微妙关系 |
| 主教/大司祭 | 意识形态、民心操控 | 大司祭的一场布道可以改变民意走向 |
| 元帅/将军 | 军事资源调配、可能政变 | 元帅卡修斯的军团是帝国最后的防线也是最大的威胁 |
| 商人/银行家 | 经济命脉、资助某一方 | 地精商会联盟的会长可以决定哪一方获得物资 |
| 阴谋家 | 信息操控、刺杀、离间 | 在皇帝耳边低声说「元帅可能不忠」的人 |
| 情报头子 | 知道所有人的秘密、真伪信息的阀门 | 帝国密探长同时掌握皇帝和元帅的黑料 |
| 继承候选人 | 还未掌权但影响各方站队 | 三个继承人背后各有一派势力提前布局 |

**关键人物的四种权力状态：**

```
1. 权实相符：名义和实际一致（如鼎盛时期的皇帝）
2. 名大于实：有头衔无实权（如衰败时的皇帝——奥瑞斯三世属于此类）
3. 实大于名：无头衔有实权（如元帅卡修斯，名义上是臣子，实际掌控军队）
4. 傀儡/影子：真正的权力在幕后（如皇太子背后的摄政太后）
```

**多种族矛盾中的关键人物：**

```
帝国圣光教会·大司祭艾琳娜（人类）
  公开立场："圣光照耀所有种族"
  私下态度：认为非人类种族是"未开化"的，但政治上不能这么说
  对种族政策的实际影响：
    - 阻止过一项歧视半兽人的法案（因为怕引发暴动，不是出于同情）
    - 暗中阻碍精灵与帝国的贸易协定（"精灵的魔法是邪术"）

矮人王国·大使钢锤（矮人）
  在帝国的困境：
    - 作为同盟大使，表面受尊重
    - 但帝国贵族在宴会上会"无恶意"地开矮人身高玩笑
    - 他的每一次忍让都在消耗矮人对帝国的耐心
  个人野心：推动帝国内部真正接纳矮人，但他的报告传回矮人王国时越来越悲观
```

**关键人物与组织状态的联动更新：**

```
事件触发 → 关键人物的决策/立场变化 → 组织状态更新 → 其他组织反应

例：
皇帝健康恶化 → 「实际权力」从5降到2 → 继承人争夺公开化 →
  → 元帅卡修斯「实际权力」从7升到9 →
  → 西境侯国叛变概率从30%升到60% →
  → 北方帝国获知情报，集结部队 →
  → 连锁反应在几个场景之间自动推进
```
