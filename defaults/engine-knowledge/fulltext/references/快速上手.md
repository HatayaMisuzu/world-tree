# 世界树引擎 — 快速上手

> 面向新用户的使用指南。
> 无需理解模块架构、隔离协议、子agent协作等内部设计，直接开始用。

---

## 最简流程（2分钟上手）

```
你：我们来写个校园故事吧
毛球：自动检测到"校园"→激活校园模块组合 + 轻量时间/随机模式
      （M1 M2 M8 M11 M12 M13 M15 M15a M16轻 M17轻）

你：创建一个角色，叫小雪，是高二转学生，性格内向但很坚强
毛球：角色"小雪"创建完成

你：场景设定在教室，开学第一天
毛球：好的，场景切换到了教室
      [世界树:校园]
      （毛球开始叙事...）
      [/世界树]
```

就这么简单。引擎会自动判断世界类型，激活合适的模块。

## 三种使用方式

### 方式一：直接开始（零配置）

直接开始叙事，毛球用默认配置快速启动：

```
"我们要讲一个关于..."
"在一个充满魔法的世界里..."
"放学后的教室里..."
```

毛球自动判断世界类型，快速开始。

### 方式二：用指令创建模组（可存档续写）

```
/引擎 new 奇幻大陆
```

毛球会依次问：
1. 世界类型 → 史诗奇幻（自动激活对应模块）
2. 初始场景和角色
3. 使用的风格预设

完成后自动进入世界树模式，叙事内容用 [世界树:] 标记包裹。
退出时自动存档，下次 `/引擎 load 奇幻大陆` 恢复。

**方式三：先搭好设定再开始**

```text
/世界书集 create 奇幻大陆 史诗奇幻
/世界书 add 魔法体系: 魔法分为三系...
/角色 create 艾琳
/场景 new 王都城门
/预设 use epic
开始吧
```

**小步迭代建议：** 先创建5-10条核心条目（世界法则+主要角色+关键地点），跑一轮叙事看看效果。不用一上来就写几十条——AI在叙事中自然提到的新概念随时可以追加。跑几轮再扩，效率和效果都更好。

## 常见场景速查

### 校园日常

```
你：/引擎 new 我的高中
    场景：教室，午休时间
    角色：我（普通学生）
    风格：normal
    
    然后开始讲你在学校的一天...
```

特点：物理规则严格，不能飞不能瞬移，但有校园日常的轻松感。
可加角色：同学、老师、社团成员。
M16/M17 轻量模式：简单的时间推进+小趣味事件。

### 奇幻冒险

```
你：/引擎 new 奥瑞利安传说
    场景：王都城门，黄昏
    角色：艾琳（流浪剑士）
    风格：epic
    
    开始冒险！
```

特点：全模块激活，规则严格但有魔法体系。
可加：组织（王国/公会）、种族（精灵/矮人）、魔法规则。
子agent自动启动（W+C+S三层并行分析）。

### 即兴创作（不创建模组）

```
你：讲个都市爱情故事，两个人在雨天的咖啡馆相遇
毛球：自动判断为"现代都市"
      [世界树:都市故事]
      （咖啡店的暖光里，你推门而入，风铃轻响...）
      [/世界树]
```

特点：快速启动，不持久化到硬盘，适合试灵感。
如果需要保留这个故事，随时可以 `/引擎 new 都市故事` 转为命名模组。

## 常用操作一览

### 第一次必用的命令

```
/引擎 new 模组名    新建故事世界
/角色 create 名字    创建角色
/世界书 add 词: 描述  添加世界观设定
/场景 new 场景名     切换到新场景
/预设 use 预设名     切换写作风格（normal/tense/epic/cozy）
开始吧              开始叙事
```

### 日常叙事中最常用的命令

```
/场景 move 地点      角色移动到某个地点
/场景 time 时间      推进时间
/场景 join 角色名    新角色进场
/场景 leave 角色名   角色退场
/角色 show 名字      查看角色卡
```

### 进阶控制

```
/模块 list           查看当前激活的模块
/模块 on M3          激活动态世界状态
/模块 off M6         关闭关系网络
/规则 check 行为     预审核某个行为是否可行
/世界状态 set 名: 值  手动更新世界状态
/认知 gap            查看角色间的信息不对称
/随机 prob           查看随机事件概率
/存档                保存当前进度
/读档 存档名          读取存档
```

### 模式切换

```
/引擎 exit           退出世界树模式，回到日常对话
/引擎 load 模组名     加载已有模组，进入世界树模式
/隔离 status         查看当前隔离状态
```

## 快速分辨：你在哪个模式

| 你的感觉 | 模式 | 回复格式 |
|---------|------|---------|
| 毛球在用角色说话、描述场景 | 世界树模式 | 回复以 `[世界树:模组名]` 开头 |
| 毛球在和你聊天、回答日常问题 | 日常模式 | 没有特殊标记 |
| 毛球在帮你创建角色、改设置 | 世界树模式（指令交互层） | 没有 [世界树] 标记，但操作影响模组 |

## 新手常见问题

**Q: 我直接讲了一个故事开头，但毛球没有用 [世界树] 包裹**
A: 因为引擎自动判断为"快速开始"模式，没有创建命名模组。如果想持久化和续写，用 `/引擎 new 故事名` 转成命名模组就行。

**Q: 我提到一个角色但毛球说不在当前世界书**
A: 角色属于哪个世界视创建时的上下文。用 `/角色 list` 看看当前世界书有哪些角色。如果角色在另一个世界书，切换到那个世界书或用 `/世界书集 switch`。

**Q: 角色做了一些超出世界规则的事**
A: 毛球会提示"这个行为在当前世界不可行"。可以用 `/规则 check` 预审核，或调整规则严格度 `/规则 strictness 宽松`。

**Q: 写了一段时间想中断，之后继续**
A: 直接 `/存档` 再 `/引擎 exit` 退出。下次 `/引擎 load 模组名` 恢复，毛球会提示"检测到未完成的故事，继续吗？"

**Q: 毛球自动判断的世界类型不对**
A: 用 `/模块 preset 正确类型` 切换模块组合就行，或者 `/模块 on/off` 手动调整。

## 写作风格预设速查

| 预设名 | 适合场景 | 特点 |
|--------|---------|------|
| normal | 日常叙事 | 中性，自然 |
| tense | 悬疑/紧张 | 短句、快节奏 |
| cozy | 温馨日常 | 细节丰富、温暖 |
| epic | 奇幻/史诗 | 宏大、华丽 |
| erotic | 色色场景 | 触觉/体液/敏感（仅在用户明确要求进入色色场景时联动 erotic-writing） |
| horror | 恐怖/惊悚 | 氛围压抑、留白多 |

用法: `/预设 use epic`
