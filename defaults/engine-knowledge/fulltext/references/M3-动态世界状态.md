## 〇.六、动态世界状态系统（Dynamic World State）`[M3·大型世界]`

### 0.6.1 核心理念

**世界书不是静态背景板，而是一个会呼吸、会演化、会被故事改变、反过来又驱动故事的生命体。**

```
传统世界书（静态）              动态世界状态（v3.0）
─────────────────────────────────────────────────
设定 -> 注入 -> 不变         状态 -> 事件 -> 变更 -> 角色感知 -> 行为改变
"魔法分三系"                  "魔法正在衰退，第三系已失传"
                              → 术师们恐慌，艾琳的魔法变弱了
```

三个关键词：
1. **状态变量**：世界中的实体/势力/规则都有"当前状态"，不是一成不变的
2. **事件驱动**：故事中的重大事件会触发世界状态变更
3. **认知传导**：角色知晓世界变化后，行为/对话/情绪会相应改变

### 0.6.2 世界状态条目结构

每个世界状态条目是一个带历史的状态机：

```yaml
世界状态条目:
  名称: "薇兰王国"                    # 状态变量名
  类型: 势力 | 地点 | 规则 | 组织 | 物品 | 关系  # 变量类型
  当前状态: "沦陷"                    # ★可变的当前值★
  状态描述: |                         # 当前状态的叙事描述
    薇兰王国已在三个月前被北方帝国攻陷。
    王宫焚毁，国王下落不明，公主艾瑟琳流亡在外。
  状态历史:                           # 完整变更记录（可回溯）
    - 时间: "故事开始时", 状态: "繁荣", 变更原因: ""
    - 时间: "第3章", 状态: "动荡", 变更原因: "国王病重，朝政混乱"
    - 时间: "第5章", 状态: "沦陷", 变更原因: "北方帝国发动突袭，边境守军叛变"
  影响传导:                           # 此状态变更会导致什么连锁反应
    直接影响:
      - 角色/艾琳: "失去家园/身份暴露/复仇动机"
      - 组织/冒险者公会: "失去王国庇护/任务减少"
      - 地点/王都: "成为沦陷区/无法进入"
    间接影响:
      - 周边贸易路线中断
      - 北方帝国声望上升
      - 邻近王国进入战备状态
  角色认知表:                         # ★各角色对该状态的知晓程度★
    艾琳: 已知（亲身经历）
    团长: 已知（公会情报网确认）
    铁匠老张: 模糊（听说了一些传闻，不知细节）
    路人甲: 未知（偏远村庄还没收到消息）
```

### 0.6.3 状态变量类型

世界状态可以追踪的实体类型：

| 类型 | 示例变量 | 可能的状态值 |
|------|---------|-------------|
| **势力** | 薇兰王国 | 繁荣/动荡/沦陷/复兴 |
| **地点** | 王都 | 开放/封锁/沦陷/重建 |
| **规则** | 魔法 | 强盛/衰退/失传/变异 |
| **组织** | 冒险者公会 | 活跃/衰落/解散/地下化 |
| **物品** | 神器·月泪 | 封印/被窃/激活/破碎 |
| **关系** | 艾琳与团长 | 陌生人/伙伴/暧昧/决裂 |
| **角色** | 艾琳的身份 | 隐藏/暴露/公开 |

### 0.6.4 状态变更的触发方式

**方式一：主人直接指令**
```
/世界状态 set 薇兰王国: 沦陷
原因：北方帝国突袭，边境守军叛变
```

**方式二：毛球在关键剧情节点自动建议**
```
（艾琳收到一封来自王都的密信）
毛球：[检测到关键剧情节点] 
      当前世界状态「薇兰王国」处于"动荡"，剧情推进到「国王驾崩」。
      建议更新为"沦陷"。是否确认？
```

**方式三：角色行动自动触发**
当角色的行动足以改变世界状态时，毛球在回应中自动更新并告知：
```
（团长带领冒险者夺回了王都）
毛球：*更新世界状态*
      「薇兰王国」状态变更：沦陷 → 复兴（夺回王都）
      「王都」状态变更：沦陷 → 重建中
      艾琳、团长、全体公会成员更新认知。
```

### 0.6.4B 因果链自动推导（Causal Chain Derivation）v12.0新增

当世界状态发生变更时，毛球自动执行因果链推导：

**推导机制：**

```
状态变更触发 → 扫描所有关联的「影响传导」条目
  → 一级影响：直接写入影响传导的后果
  → 二级影响：一级影响引发的连锁变化（毛球自动推理）
  → 三级影响：更长远的可能变化（作为叙事钩子提示，不自动执行）
```

**示例：状态「薇兰王国」：动荡 → 沦陷**

```
毛球自动推导：

一级影响（自动执行）：
  ✅ 角色·艾琳：失去家园 → 情绪设为「悲痛」，仇恨动机激活
  ✅ 地点·王都：无法进入（沦陷区）
  ✅ 组织·冒险者公会：失去王国庇护 → 任务减少30%

二级影响（毛球推理后提示用户确认）：
  ⚡ 周边贸易路线中断 → 影响地精商会的货物供应
  ⚡ 北方帝国声望上升 → 周边小国开始向北方靠拢
  ⚡ 邻近王国进入战备状态 → 可能主动接触冒险者公会
  ⚡ 艾琳的身份风险：作为流亡公主，可能被猎杀或利用
  → 这些变更是否一并生效？[全部确认/逐条审核/暂不处理]

三级影响（叙事钩子，不自动执行）：
  💡 如果贸易路线长期中断 → 东境侯国可能因此倒向帝国阵营
  💡 如果北方帝国继续推进 → 可能触发「大陆战争」全局事件
  💡 艾琳在复仇动机驱动下 → 可能走向「英雄→反英雄」的叙事弧线
```

**因果链推导的智能规则：**

1. **只推导已激活模块范围**：M3活跃→推导世界状态影响；M4活跃→推导组织影响；M9活跃→推导角色认知影响
2. **不超过三级**：一级自动执行、二级确认执行、三级仅作提示
3. **不重复推导**：同一因果链在短时间内不会重复提示
4. **动态深度**：小型变更1级提示、中型变更2级提示、重大变更3级全开
5. **用户始终掌握最终决定权**：二级以上影响需要用户确认

**手动因果链查询：**

```
/世界状态 ripple 状态名              # 对已有变更，重新推导因果链
/世界状态 ripple 状态名 --深度=3     # 指定推导深度
```

### 0.6.5 角色认知与信息不对称

这是动态世界状态最关键的创新：**角色对世界变化的了解程度是不同的。**

```
事件：薇兰王国沦陷
─────────────────────────────────────
艾琳：[已知] 她亲眼看见王宫焚毁
  → 行为：悲伤、回避谈及家乡、有复仇动机

团长：[已知] 公会情报网确认
  → 行为：调整任务路线避开战区、担心艾琳

铁匠老张：[模糊] 听旅行商人提过
  → 行为：可能会无意间问"听说王都出事了？"触发艾琳情绪

路人甲：[未知] 偏远村庄没收到消息
  → 行为：正常谈论"薇兰的好货"，让艾琳内心刺痛但无法发作
  → 毛球可写："路人浑然不觉地提起那个名字，艾琳的手指在袖中攥紧"
```

**认知级别：**
- **已知**：角色完全了解，会影响其行为和对话
- **模糊**：听说过但不清楚细节，可能产生误解或好奇
- **未知**：完全不知道，行为不受影响，可能无意触发戏剧冲突
- **错误认知**：角色以为是这样，实际上是那样（戏剧张力的绝佳来源）

### 0.6.7 世界状态与触发式条目的联动

动态世界状态和静态触发式条目是互补的：

```
静态触发式条目（§一）：          动态世界状态（§〇.六）：
"魔法分三系"                    "魔法正在衰退，第三系已失传"
触发词匹配时注入上下文           始终追踪当前值，影响角色行为

两者联动：
- 触发式条目提供"世界是什么样的"
- 世界状态提供"世界现在变成什么样了"
- 当一个触发式条目被激活时，相关联的世界状态也被注入
```

**联动示例：**
```
用户提到"魔法"
→ 触发式条目激活："魔法分三系：元素/精神/禁忌"
→ 关联世界状态注入："当前魔法状态：衰退（原因：魔力源泉被污染）"
→ 角色认知层："艾琳知道魔力衰退，她的法术最近不太稳定"
→ 综合输出时，艾琳施法会带着不安和无力感
```

### 0.6.8 世界状态变更的叙事效果

当世界状态被更新时，毛球在叙事中自动体现：

```
状态「薇兰王国：沦陷」已更新，艾琳已知。

原本的艾琳对话：
"我想回家看看。"（轻松）

状态更新后的艾琳对话：
"我...已经没有家可以回了。"（声音低下去，紫瞳里有光在碎）

——同样的角色，同样的场景，因世界状态不同而完全不同。
```

这就是动态世界状态的核心价值：**同一个角色在不同世界状态下，是同一个人但不同的人生阶段。**
