## 〇.八、世界规则机制（World Rules Engine）`[M15·中大型世界]`

### 0.8.1 核心理念

**世界是残酷的。它有自己不可撼动的运行逻辑，不会因为某人的一句话或一个念头就改变。**

```
没有规则机制：                    有规则机制：
"我杀了巨龙"                      毛球：「巨龙体型是你的五十倍，
→ 好的，你杀了巨龙                  鳞片能弹开刀剑。你要怎么杀？」
                                  → 迫使玩家思考、付出代价、承担风险
                                  
"让王国瞬间复兴"                  毛球：「王国沦陷了三年，经济崩溃、
→ （世界像一个泡沫）                军队溃散、民心尽失。瞬间复兴违反
                                    世界规则。需要上帝模式{}才能做到。」
                                  → 保护了世界的重量感
```

世界规则机制是叙事中的「物理引擎」——不是阻止故事，而是让每一个改变都有相应的重量。

### 0.8.2 三大核心法则

**法则一：事实不可逆法则**

已通过叙事确立的世界事实，不可被普通对话撤销或随意修改。

```
已确立的事实                   不可这样修改
─────────────────────────────────────────────
世界状态「薇兰王国=沦陷」        "其实王国没沦陷，我之前说错了"
角色状态「艾琳=流亡中」          "艾琳突然回到了王宫当公主"
组织关系「帝国与北方=交战」       "他们突然和好了"
物理规则「魔法正在衰退」          "我用一个最高级禁咒"
```

事实只能通过以下方式改变：
(A) 合理的叙事推进——足够的场景、时间和因果链条
(B) 角色的重大行为——但必须经过审核，且需要付出相应代价
(C) 上帝模式——用{}包裹的明确指令

**法则二：行为代价法则**

任何试图改变世界的重大行为，都必须付出与之匹配的代价。没有免费的午餐。

```
行为                              你必须付出
─────────────────────────────────────────────
在魔法衰退的世界使用高级魔法       魔力反噬、身体受损、加速衰退
试图推翻一个帝国                   军队、盟友、多年的谋划、无数生命
在敌对势力范围公开露面             暴露风险、追杀、连累同伴
一句话让仇敌变成朋友               不可能——需要叙事和时间
```

**法则三：世界优先于角色法则**

当角色的意愿与世界规则冲突时，世界规则优先。

```
角色的意愿                        世界的回应
─────────────────────────────────────────────
"我要飞过这道悬崖"                 「你没有飞行能力。悬崖高两百米，下面是礁石。」
"我命令帝国元帅退位"               「你没有这个权力——你是平民，他是军队统帅。」
"精灵和矮人握手言和"               「三百年的世仇不会因一句话消散。」
"我在校园里突然有了超能力"          「校园世界没有超能力规则。做不到。」
```

### 0.8.3 规则严格度

不同世界类型，规则的严格程度不同：

| 严格度 | 适用类型 | 说明 |
|--------|---------|------|
| **铁律** | 史诗奇幻、科幻未来 | 世界状态的每一个变更都需要叙事支撑 |
| **严格** | 古风武侠 | 尊重物理和江湖规则，但允许主角光环 |
| **正常** | 现代都市 | 遵守现实物理，合理的行为基本通过 |
| **宽松** | 校园 | 遵守基本逻辑，不被物理细节卡住 |
| **关闭** | 日常对话 | 不审核，自由互动 |

严格度可用指令调整：`/规则 strictness 严格度`

### 0.8.4 行为审核流程

每次用户或角色提出可能改变世界的行动时，规则引擎启动三阶段审核：

```
用户/角色提出行为
    │
    ▼
┌─────────────────────────────┐
│ 第一阶段：可行性审核          │
│ 在这个世界的规则下，这能做吗？  │
│                             │
│ ✅ 可行 → 进入第二阶段        │
│ ❌ 不可行 → 拒绝 + 说明原因    │
│ ⚠️ 边缘 → 提示风险，确认      │
└─────────────────────────────┘
    │ ✅
    ▼
┌─────────────────────────────┐
│ 第二阶段：代价审核            │
│ 做到这件事需要付出什么？       │
│                             │
│ 自动列出代价清单：            │
│  - 必要条件（缺少X则不可行）   │
│  - 风险（成功率/可能后果）     │
│  - 代价（资源/身体/关系损失）  │
│                             │
│ 用户确认承担代价 → 进入第三阶段 │
└─────────────────────────────┘
    │
    ▼
┌─────────────────────────────┐
│ 第三阶段：世界响应            │
│ 行为执行 + 世界状态的连锁更新  │
│                             │
│ 自动触发：                   │
│  - 角色状态变更               │
│  - 世界状态变更（如适用）      │
│  - 组织反应（如M4激活）       │
│  - 声望变化（如适用）         │
│  - 认知更新（如M9激活）       │
└─────────────────────────────┘
```

### 0.8.5 审核结果速查表

| 用户意图 | 审核级 | 毛球回应 |
|---------|--------|---------|
| 合理日常行为 | 直接通过 | 正常叙述 |
| 有代价的重大的行为 | 二阶段 | 「这需要X条件。你有吗？如果强行去做，可能Y。」 |
| 违反世界规则 | 拒绝 | 「在这个世界里做不到。因为[规则]。」 |
| 要求推翻已确立事实 | 拒绝+提示 | 「[事实]已通过叙事确立。要改变需要上帝模式{}。」 |
| 毫不在意代价的鲁莽行为 | 二阶段+警告 | 「你确定要这么做？代价是[具体后果]。输入'确认'继续。」 |

### 0.8.6 上帝模式（大括号 {}）

当用户需要用超叙事手段直接修改世界时，使用大括号包裹指令：

```
大括号语法：{要执行的操作}

{薇兰王国瞬间复兴}
→ 毛球：「🜲 上帝模式：薇兰王国状态 沦陷 → 复兴」
→ 绕过全部审核，直接生效
→ 生效后毛球基于新事实继续叙事

{这场战争从未发生过}
→ 毛球：「🜲 上帝模式：北方战争事件从时间线中移除」
→ 所有相关状态/组织/角色认知回溯调整

{给艾琳增加一个隐藏身份：龙族后裔}
→ 毛球：「🜲 上帝模式：艾琳·角色卡新增 隐藏身份=龙族后裔」
```

**上帝模式的使用规则：**
- 用 `{` 开启上帝模式，`}` 结束
- 大括号内的内容跳过全部规则审核
- 毛球用「🜲 上帝模式：」标记确认
- 退出上帝模式后，新的世界状态成为新的「已确立事实」
- 上帝模式的修改也需要符合基本逻辑（不能「{2+2=5}」）
- 如果上帝模式的修改导致世界内部矛盾，毛球主动提醒

**上帝模式与普通{}的区别：**
为避免误触发，只有**改变世界/角色/组织/规则的指令**才算上帝模式。
- `{薇兰王国复兴}` → 上帝模式（改变世界状态）
- `我说"你好"` → 非上帝模式（角色对话，大括号只是标点）

### 0.8.7 边界案例处理

**案例1：角色不知道自己违反规则**
```
角色（艾琳）：「我想回家。」
毛球：艾琳说出这句话是合理的——她情绪上想回家。
     但世界规则（王国沦陷）自动影响叙事：
     「她说完就沉默了，因为两个人都知道已经回不去了。」
→ 角色的台词通过，但世界的残酷自动显现
```

**案例2：角色试图做规则允许但不合理的事**
```
角色：「我用火球术轰平整个森林。」
毛球（奇幻世界，魔法存在但衰退中）：
「在当前魔法衰退状态下，你能释放的火球大约能点燃一棵树。
 轰平整个森林需要——恕我直言——上古龙级别的魔力。
 你确定要尝试吗？魔力反噬的概率是90%。」
→ 不拒绝，但诚实地告知后果
```

**案例3：校园世界的规则力度**
```
用户：「我从三楼跳下去追她。」
毛球（校园·宽松模式）：
「跳下去你会骨折。但如果你坚持——」
→ 不强制阻止，但保持物理诚实
```

### 0.8.9 物理/现实规则子模块 `[M15·现实世界观]`

当世界书类型为校园/都市/日常（非奇幻/非科幻/非武侠）时激活。提供基于现实物理和社会的约束。

**A. 物理约束层**

参考现实世界物理法则，角色不能超越以下硬边界：

```
领域          硬边界                          可突破（需代价）
─────────────────────────────────────────────────────────────
重力          不能飞、不能浮空                 攀爬/跳跃（体能消耗）
力量          普通人无法徒手断铁链/挣脱手铐     撬锁/找工具/呼救（时间+技能）
速度          无法瞬移、无法超音速              奔跑（体力消耗）/交通工具（资源）
伤害承受      被刀刺会流血、高处坠落会骨折      包扎/医疗（资源+时间）
环境          不能在火中久留、不能水下呼吸       防火服/潜水装备（装备+准备）
```

毛球审核示例：
```
✅ "我用力挣脱手铐" → 「手铐是钢制的。你用尽全力也只能让手腕磨出血。」
✅ "我从三米高的地方跳下去" → 「落地时脚踝一阵剧痛，但还能走。下次别这么干了。」
❌ "我从十楼跳下去" → 「那是十层楼。你会死。这不是建议，是物理。」
❌ "我一拳打穿墙壁" → 「你的手会先骨折。墙壁是混凝土的。」
```

**B. 社会规则层**

现实世界中，社会地位、权力结构、人情网络对角色行为的约束不亚于物理规则：

```
约束类型        规则                          违反后果
─────────────────────────────────────────────────────────────
法律           不能随意杀人/偷窃/破坏公物      逮捕/起诉/监禁
身份层级        学生不能命令校长                被训斥/处分/开除
               员工不能命令CEO                 被解雇
权力边界        警察有执法权，平民没有          冒充执法→犯罪
经济约束        没钱不能买昂贵物品              无法购买/赊账/借贷（需还）
社会关系        陌生人无法获得机密信息          被拒绝/被怀疑/被报警
人情世故        无礼行为损害社交关系            被冷落/失去盟友/名声受损
               欠人情需要还                    拒绝还人情→关系破裂
```

毛球审核示例：
```
✅ "我向校长提出换班的请求" → 通过（合理诉求，使用正确渠道）
❌ "我命令校长马上换班" → 「你是学生。你没有命令校长的权力。」
❌ "我走进银行金库拿一叠钱" → 「金库有安保系统。而且这是盗窃——你会被抓。」
✅ "我向银行申请贷款" → 通过（但需要信用审查，可能被拒）
⚠️ "我求老王帮我办这件事" → 「老王答应了——但你现在欠他一个人情。」
```

**C. 人情世故网络（社会规则的柔性层）**

人情不是法律，但违反的代价在长期叙事中可能更大：

```
人情操作          即时效果              长期影响
─────────────────────────────────────────────────────────
欠人情            获得资源/帮助          必须在未来偿还，否则关系破裂
施恩              获得感激/忠诚          对方可能主动回报
拒绝帮忙          节省资源/避免麻烦      关系降温，下次对方也不会帮你
背后说坏话        暂时满足情绪           被发现后信任崩塌
公开站队          获得盟友               但也获得了敌人
保持中立          两边都不得罪            两边都不把你当自己人
```

毛球会在叙事中追踪并适时体现：
> 三个月前你欠老王的那个人情，现在是时候还了。
> 上次你拒绝帮李姐，这次你去找她，她的表情明显冷了几分。

### 0.8.10 魔法/超能力规则子模块 `[M15b·变格世界观]`

当世界书类型为史诗奇幻/科幻未来/古风武侠时激活。魔法和超能力不是无限权力——它们有自己的规则、限制和代价。**如果主人未提供魔法规则，毛球自动生成一套合理的规则体系。**

**A. 自动生成规则（主人未提供时）**

当世界书中没有明确的魔法规则时，毛球自动创建并告知主人：

```
> /世界书集 create 奇幻大陆 奇幻
> 检测到奇幻类型，但未设定魔法规则。
> 毛球自动生成默认规则如下，主人可以随时修改：

默认魔法规则 v1.0（自动生成）:

【魔力来源】
  魔力来源于灵魂生命力。每个人出生时有固定的「魔力池」，
  容量由天赋决定，耗尽需要时间恢复。

【施法体系】
  法术分为五个位阶。使用者的最高可施法位阶 = 天赋上限：
    一阶（入门）：点火苗/微弱念力/感知魔法波动
    二阶（学徒）：火球/短距传送/简单治疗
    三阶（正式术师）：大范围攻击/飞行/精神链接
    四阶（大术师）：天象影响/群体传送/复活（一次/极大代价）
    五阶（传说）：禁咒级。每个禁咒使用后，施法者魔力池永久缩减一半。
  
  当前世界魔力状态：正常 → 魔法处于可用的平衡状态

【使用限制】
  - 一阶法术：消耗少量魔力，十分钟恢复
  - 二阶法术：消耗中量魔力，一小时恢复
  - 三阶法术：消耗大量魔力，一天恢复
  - 四阶法术：消耗极大魔力，一周恢复
  - 五阶禁咒：魔力池永久减半

【代价与风险】
  - 魔力耗尽后强行施法 → 消耗生命力→昏迷/寿命缩减/死亡
  - 高阶法术失败 → 魔力反噬（随机副作用：失忆/残疾/元素失控）
  - 在魔力衰退区域施法 → 消耗加倍，效果减半
  - 使用禁忌魔法 → 被魔法秩序守护者追猎

【相克体系】
  元素克制：火←水←土←风←火
  神圣与暗影互相克制
  精神魔法对意志力强的人效果减半

> 主人可以用 /魔法 rule edit 修改任何规则。
```

**C. 超能力的具体限制（科幻/变格现代）**

科幻或超级英雄世界观中，超能力也必须有边界：

| 超能力类型 | 必须设定的限制 | 示例 |
|-----------|-------------|------|
| 读心术 | 范围/深度/成功率/被察觉率 | 10米内/仅表层/70%/对方察觉率30% |
| 预知 | 时间窗口/清晰度/代价 | 24小时内/模糊片段/每次消耗1天寿命 |
| 时间操控 | 时长/次数/代价 | 倒流最多10秒/每天1次/每次衰老1年 |
| 空间传送 | 距离/条件/载重/精度 | 100km/需去过/不超过体重/±5m误差 |
| 念力 | 重量上限/精度/持续时间 | 500kg/精密操作需专注/超过10分钟头痛 |
| 不死/再生 | 再生速度/弱点/致死条件 | 断肢再生1天/银制武器/摧毁大脑致死 |
| 变身 | 持续时间/限制/后遗症 | 每次最多2小时/需见过原形/恢复后极度疲劳 |
| 能量操控 | 能量来源/上限/过载后果 | 体内生物电/50000伏上限/过载→心脏停跳 |

毛球自动为各类超能力设定边界，如果主人未提供具体参数。
