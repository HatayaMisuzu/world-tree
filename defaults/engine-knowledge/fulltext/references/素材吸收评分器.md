# 素材吸收评分器

> world-tree v12.19 创新升级模块。定位：素材进入世界树前的污染风险与可迁移性评分。

## 评分维度

| 维度 | 说明 | 结果 |
|------|------|------|
| 技法价值 | 是否能提取写作技法 | 高/中/低 |
| 事实污染 | 是否会把外部设定误写入本世界 confirmed | 高/中/低 |
| 风格污染 | 是否会覆盖当前模组语气 | 高/中/低 |
| 触发价值 | 是否适合转成世界书条目 | 高/中/低 |
| 来源清晰 | 是否能回溯到来源文件/段落 | 是/否 |

## 处理结果

| 结果 | 行为 |
|------|------|
| absorb | 可吸收为技法或候选条目 |
| quarantine | 进入隔离池，需用户确认 |
| reject | 不吸收，仅记录原因 |

## 建议格式

```json
{
  "source": "素材文件名",
  "decision": "quarantine",
  "technique_value": "高",
  "fact_pollution": "中",
  "style_pollution": "低",
  "reason": "技法可迁移，但含大量外部专名，不能直接进入当前世界书。"
}
```

## 硬规则

- 外部素材只能先进入候选池，不得直接进入 confirmed。
- 原文引用必须保留来源文件和片段位置。
- 可迁移的是技法，不是未经确认的外部事实。
