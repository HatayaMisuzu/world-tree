# 世界树引擎 — 变更日志

## 2026-06-01 — 控制台会话管理与体检升级

- 对话页新增「会话管理器」：按模组保存最近 Hermes session，支持手动切换、复制 ID、忘记绑定和清除当前绑定。
- 新 session 首次发送消息时自动附带启动包，内容包含当前模组、活跃分支、当前场景、最近摘要链、在场角色与待决问题，减少新对话缺上下文的问题。
- 模组聊天键改为 `storagePath/storageKey + hash` 稳定指纹，不再只依赖模组名，避免同名世界书共享会话与本地聊天记录。
- 新增「体检」Tab：显示控制台版本、HTML hash、API 连通状态、目录记忆权限、已知会话和主 / writer 镜像目标路径。
- 更新文档：补充会话管理器、启动包、体检页和稳定模组键的说明。
- 验证：控制台脚本语法检查通过；后续同步主档案与 writer 档案后再跑 `quick_validate.py`。

## 2026-06-01 — 控制台存档交互升级

- 存档页改为按世界书/模组分组展示已有存档，支持跨模组搜索存档、场景和摘要。
- 新增「手动存档」按钮：生成 `/存档 <名称>` 指令，复制到剪贴板并填入对话输入框。
- 每条存档新增「加载」「替换」「删除」操作：分别生成 `/读档 <名称>`、`/存档 <名称>`、`/存档 delete <名称>` 指令；非当前模组的存档会自动在指令前追加 `/引擎 load <模组>`。
- 控制台仍保持只读边界：所有存档写入/删除/替换都通过 Hermes 指令执行，不直接改 JSON 文件。

验证：主档案与 writer 档案控制台已同步；两边 `quick_validate.py` 均通过。
