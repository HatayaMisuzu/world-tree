# Install And First Run

Status: Productization Closure PARTIAL.

## Install

```bash
npm install
```

## Start

```bash
npm start
```

Open the local console at:

```text
http://127.0.0.1:3000
```

If the server prints another port, use the printed local URL.

## First Run

Recommended current path:

1. Open the console.
2. Use Quick Setting or Creation Forge / Alchemy.
3. For Creation Forge, enter an idea or setting.
4. Generate a creation map.
5. Select final output targets.
6. Generate preview.
7. Generate local folder draft.
8. Confirm delivery.
9. Open the created world and send one play turn.
