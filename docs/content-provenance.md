# Content Provenance

本文件记录随开源仓库分发的默认内容来源决策。

## 2026-06-12

用户决策：`defaults/` 下来源不明或版权风险较高的可读内容基本全部移除。

已移除并不随开源仓库分发：

- `defaults/engine-knowledge/`
- `defaults/cases/`

保留内容：

- `defaults/engine-profile/`：运行时模块配置与知识卡，属于项目自身引擎配置。
- `defaults/world-profiles/`：内置模式配置，属于项目自身配置。
- `defaults/examples/manifest.json`：空素材清单，仅作为后续登记示例世界或角色卡的入口；当前不随仓库分发示例故事、角色卡或案例素材。

后续如需重新加入示例世界、案例或知识库内容，必须在本文件记录来源：原创 / 已授权 / 公共领域 / 许可证兼容，并附上必要说明。
