# Experience Stability Kernel P1

P1 consumes the P0 Living World Packet and adds five bounded stability mechanisms:

- deterministic impact classification and stop-loss reverse proposals;
- mode-aware context routing, deep secret filtering, and hard per-source budgets;
- runtime-first emotional inertia with one-step enum transitions;
- plan-only Director output guarded against reveals and player-choice substitution;
- seed/sprout/branch/pruned worldbook candidates, with branch promotion gated by proposal approval.

Critical proposals require a second confirmation. Major and critical approvals open three-turn and five-turn stop-loss windows respectively. Reversal is a new proposal; original tracking remains intact.

The Experience Stability Packet is read-only with respect to canon. It returns a bounded context packet, emotional summaries, a guarded Director plan, impact policy, and a Growth Tree snapshot.
